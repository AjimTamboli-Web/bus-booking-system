# 🚌 Bus Booking System (Java + JDBC + Multithreading)

## 📌 Project Description
This is a console-based Bus Booking System built using Java.  
It simulates real-world booking, boarding, and dropping of passengers with database integration.

---

## 🚀 Features
- Multi-threaded seat booking
- Real-time bus movement simulation
- Passenger boarding & dropping logic
- JDBC integration with MySQL
- DAO + Service Layer Architecture
- Console-based UI

---

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL
- Multithreading

---

## 📂 Project Structure

dao → Database operations
model → Entity classes
service → Business logic
thread → Booking threads
main → Console UI


---

## ▶️ How to Run
1. Clone repository
2. Setup MySQL database
3. Update DB credentials in `DBConnection.java`
4. Run `ConsoleApp.java`

---

## 🧠 Learning Outcomes
- Thread synchronization
- JDBC operations
- Layered architecture (DAO, Service)
- Real-world backend simulation
